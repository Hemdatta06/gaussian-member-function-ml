# Gaussian Membership Function (ML Lab)

This project demonstrates the implementation of a Gaussian Membership Function using Python.

## Description
The program uses student height data to compute the mean and standard deviation, and visualizes the data using:
- Histogram (frequency distribution)
- Gaussian membership function curve (values between 0 and 1)

## Objective
To understand how Gaussian functions are used in fuzzy logic to represent membership values based on data distribution.

## Steps Performed
- Collected height data of students
- Calculated mean and standard deviation
- Defined Gaussian membership function
- Plotted histogram of data
- Plotted Gaussian curve using a secondary axis

## Tools and Libraries
- Python
- NumPy
- Matplotlib

## Output
- Histogram representing frequency distribution of heights
- Smooth Gaussian curve showing membership values

## Conclusion
The Gaussian Membership Function provides a smooth and continuous way to represent data distribution and is widely used in fuzzy systems.
